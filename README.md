## Fora

[Fora](https://fastforge.github.io/fora/) is just a toy project written in C++.

### Dependancies

`Boost` will be needed for building. Platform:Ubuntu 1604-amd64

### Compilation
instructions:
```
cd fora
mkdir build
cd build
cmake ..
make
```
